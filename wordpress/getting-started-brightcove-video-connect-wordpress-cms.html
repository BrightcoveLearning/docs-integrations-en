Releases

All releases of the Video Cloud WordPress connector can be found on the Releases page. We recommend that you use the latest release unless you need an older one to support the version of WordPress you are using. The current verion of the connector supports WordPress 6.4.

Documentation

For instructions on installing and using the plugin, go to the Brightcove Video Connect Readme page.

Getting help

The Brightcove Connector for WordPress is an open source plugin. You are free to install & use it for your needs. If you are unable to successfully install the connector on your WordPress server, Brightcove offers a CMS Connector onboarding package which provides direct access to the connector developers for installation support. Your Brightcove Account Manager can provide more details on the onboarding package.

If the Brightcove Connector is installed but not working as documented, please fill out this support worksheet and include it with a support ticket to Brightcove customer support.
