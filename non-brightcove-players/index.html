  Playback API

The Video Cloud platform supports third-party players. For the integration with third-party players, the Playback API must be used to get the manifest file to the video.

  • Quick Start: Playback API
  Data Collection API

If a third-party player is used, the Data Collection API is also necessary to report analytics, at least to calculate the number of streams.

  • Overview: Data Collection API v2
