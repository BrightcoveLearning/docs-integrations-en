Installing the player plugin

To integrate Kollective with Brightcove Player, add the Kollective plugin to your player.

The Kollective Plugin is supported with the Brightcove Player version v6.58+.

The following steps detail using Video Cloud Studio to load the plugin.

  1. In Video Cloud Studio, open the PLAYERS module and either create a new player or locate the player to which you wish to add the plugin.
  2. Select the link for the player to open the player's properties.
  3. Select Plugins in the left navigation menu.

  4. Next, select the Add a Plugin button, and then select Brightcove Plugin.

  5. Expand the Brightcove Plugin dropdown and select kollective.

  6. For the Options(JSON): Enter the following:

    • ksdkUrl - Points to the ksdk library from Kollective -- use the version of your choice
    • It is including the the URL and policy key as part of the plugin configuration:
      {
    "ksdkUrl": "https://cdn.kollective.app/sdk/ksdk-latest.min.js",
    "policy_key": "123456789"
  }
  7. Select the Save button.
  8. Publish your player.
