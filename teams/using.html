Introduction

Before using the Brightcove Media Manager, it must be installed and configured for the team. See Installing the Connector for Teams for details.

After you have added the Media Manager to your Team as explained in the previous section, you may add and manage videos add and manage playlists and manage experiences from the Media Manager within Teams.

The changes you make from the media manager within Teams gets saved to your Brightcove account on the Brightcove platform and you are able to see them when you connect via Brightcove Studio online.

Add videos

  1. Open Teams and navigate to the Team and Channel where you previously installed the Media Manager.
  2. Select the Asset Manager tab to open the Media Manager.
  3. Click the Add Videos tab:
  4. Either drag and drop the video you want to upload on the indicated hot-spot, or click Browse for Videos to select the video to add:
  5. Once it is added, the video gets processed on the Brightcove video cloud platform. Wait for the processing to complete.
  6. The video is ready to edit/manage when the processing is done and you may click on the Edit Video link as shown below:
  7. You will be navigated to the edit page for the video where you can then edit the video metadata.

Manage videos

While in the Media Manager, click the Manage Videos tab.

Edit a video

  1. Find the video you want to edit from the list of videos.
  2. Click the Edit link:
  3. Edit the metadata for the video as desired:
  4. When you are finished, scroll to the bottom of the edit page and click the “Update on Video Cloud” button to save your changes to the Brightcove Video cloud:

Get video embed code

  1. Find the video you want the embed code for from the list of videos.
  2. Click the Embed button:
  3. In the Embed Video modal, click the Copy to Clipboard link to copy the embed code.
    Note: you can set various configs for the embed to govern its behavior, and also preview it before you copy the code, if desired.

Add playlists

  1. In the Media Manager, click the Manage Playlists tab.
  2. Click the Add New Playlist button at the bottom of the list of playlists:
  3. Enter a name and description, and select the videos you want as part of the playlist.
  4. Click the Create button:
  5. The playlist gets processed by the Brightcove Video cloud and is displayed in the list of playlists after processing
    Note: processing may take a few moments and the playlist may not be immediately visible on the list of playlists – if desired, confirm processing using the Brightcove Studio.

Manage playlists

In the Media Manager, click the Manage Playlists tab.

Edit a playlist

  1. Find the playlist you want to edit.
  2. Click the Edit link.
  3. Make your edits and click the Save button.

Get playlist embed code

  1. Find the playlist you want to get the embed code for.
  2. Click the Embed button.
  3. In the Embed Playlist modal, click the Copy to Clipboard link to copy the embed code.
    Note: you can set various configs for the embed to govern its behavior, and also preview it before you copy the code, if desired.

Manage experiences

In the Media Manager, click the Manage Experiences tab.

Preview experience

  1. Find the experience you want to preview.
  2. Click the Preview link.

Get experience embed code

  1. Find the experience you want to get the embed code for.
  2. Click the Embed button.
  3. In the Embed Experience modal, click the Copy to Clipboard link to copy the embed code.
    Note: you can set various configs for the embed to govern its behavior, and also preview it before you copy the code, if desired.

Add a video to a teams channel tab

  1. Open Microsoft Teams and navigate to the Team you want to add a video on.
  2. Open the channel where you want to add the video.
  3. Click the Add a tab menu:
  4. Select the Brightcove Video Connector app from the list of available apps:
  5. In the configuration modal, select the Brightcove Account.
  6. In the configuration modal, select Embed Video as the connector tab mode and click Next:
  7. Select the video you want to add to the channel tab and click Save:
  8. You should now see the added video on the channel tab.

Add a experience to a teams channel tab

  1. Open Microsoft Teams and navigate to the Team you want to add a experience on.
  2. Open the channel where you want to add the experience.
  3. Click the Add a tab menu:
  4. Select the Brightcove Video Connector app from the list of available apps:
  5. In the configuration modal, select the Brightcove Account.
  6. In the configuration modal, select Embed Experience as the connector tab mode and click Next:
  7. Select the experience you want to add to the channel tab and click Save:
  8. You should now see the added experience on the channel tab.
