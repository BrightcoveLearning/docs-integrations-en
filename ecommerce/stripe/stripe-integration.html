Configuring Stripe Connect

  1. Enter the Stripe Dashboard.
  2. Navigate to the Connect menu and Click the Create button.
  3. Select the Country for the desired platform and the Stripe Connect Account type.
  4. A one-time link will be generated, where you can follow the onboarding process.

Onboarding page

  • After filling out your onboarding data, your Stripe Connect account will show up on the connect entries.
    Everything from Products, prices etc must be configured inside the connected account.

Identifying the Connected Account ID

Upon entering the Connected Account, click on the Copy ID button to get your Connect Account ID. That ID will be required for the stripe configuration.

Afterwards, navigate to Developers -> Webhook, to check if a webhook for connected accounts has been set up.

Connected Accounts Webhook

On the overview page, the connect account webhooks will be of the type Connect. After finding the desired webhook, enter it to confirm it supports the required events listed below. If not, those events must be added to that webhook, or a new one must be created. Below, you'll find how to register a new webhook related to stripe-connected accounts.

Registering a new Webhook

The new webhook will have a list of events, that will be fired by stripe to our webhook every time they are triggered.

Current supported events:

  • 'customer.subscription.deleted'
  • 'customer.subscription.updated'
  • 'customer.subscription.created'
  • 'customer.subscription.trial_will_end'
Any other events registered will be ignored by our webhooks.

On the creation page, fill in our endpoint URL.

In the Listen to section, be sure to select the Events on Connected accounts option and insert the events listed above in the Select Events section.

Getting the Webhook Signing Secret

To get the signing secret, enter the webhook page and click on the Signing Secret section to see the code. This code is required to configure the connect account in the Stripe Payment Gateway API.
