Introduction

It is recommended to use a Standard Stripe connect account.

Configuring Stripe Connect

  1. Enter the Stripe Dashboard.
  2. Navigate to the Connect menu and Click the Create button.
  3. Select the Country for the desired platform, and the Stripe Connect Account type.
  4. A one-time link will be generated, you can onboard through there.

Onboarding page

  1. After the customer fills up their onboarding data, their Stripe connect account will show up on the connect entries. Everything from Products, prices etc must be configured inside the connected account.

Identifying Connected Account ID

Upon entering the Connected Account, click on the “Copy ID” button (or check the ID section of the account) to get the customer’s Connect Account ID. That ID will be required for the stripe configuration.

Afterwards, we head over to Developers->Webhook, to check if there’s already a webhook for connected accounts set up.

Connected Accounts Webhook

On the overview page, the connect account webhooks will be of the type Connect. If there’s already one, enter in it to confirm it supports the required events listed below. If not, those events must be added to that webhook or a new one must be created. Below, we’ll list on how to register a new webhook related to stripe connected accounts.

Registering a new Webhook

The new webhook will have a list of events, that will be fired by stripe to our webhook every time they are triggered.

Below are the list of our current supported events:

  • 'customer.subscription.deleted'
  • 'customer.subscription.updated'
  • 'customer.subscription.created'
  • 'customer.subscription.trial_will_end'
Any other events registered will be ignored by our webhooks.

On the creation page, fill in our endpoint URL, On the Listen to section, be sure to select the Events on Connected accounts option and insert the events listed above on the select events section.

Getting the Webhook Signing Secret

To get the signing secret, enter in the webhook page, and click on the Signing Secret section to see the code. This code is required to configure the connect account in the Payment API Stripe Payment Gateway.

After we have both the Connected Account ID and the Webhook Signing Secret, we can configure it properly in the Managing Payment Gateway configurations.
